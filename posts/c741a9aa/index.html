使用 FastGithub 稳定的访问 GitHub

安装使用教程

项目地址：dotnetcore/FastGithub: github 加速神器，解决 github 打不开、用户头像无法加载、releases 无法上传下载、git-clone、git-pull、git-push 失败等问题

如果上面的地址打不开，这里还准备了备用的 FastGithub 下载地址。选择你需要下载的安装包然后进行解压，建议选择 最新版 的，我选择的是 fastgithub_win-x64.zip，解压后双击打开 FastGithub.UI.exe，不出意外的话你的 GitHub 就能正常访问了。

FastGithub 已经寄了，我现在更推荐使用 docmirror/dev-sidecar 来平替，可惜 Gitee 上的同步项目已经被封禁了，目前只能在 Github Release 上下载了。

可笑的是，如果你连 GitHub 都进不去的话，那就更别提在上面下载辅助工具了，唉听我的，实在不行咱就整个梯子吧，我也没有其他更好的办法了，恕我爱莫能助。
